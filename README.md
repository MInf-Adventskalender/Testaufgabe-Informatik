# Testaufgabe-Informatik
In der Datei <i>code.py</i> ist ein Programm vorhanden. </br>
Es werden 3 Zahlen <code>a, b, c</code>, getrennt mit Leerzeichen, eingegeben. Dabei soll nichts geändert werden. </br>
In <code>res</code> soll der Wert von <code>a * b + c</code> gespeichert werden. Ändere dafür den Code auf Zeile 6. </br>
Dabei gilt Punkt vor Strich und folgendes: </br>
\+ Addition </br>
\- Subtraktion </br>
\* Multiplikation </br>
\/ Division </br>
\() Klammern </br>
